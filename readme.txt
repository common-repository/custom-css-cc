Enable to add custom CSS via admin.

== Description ==

The plugin enable to add CSS in option page. The CSS of the plugin overide CSS of theme,it's enable to edit style without to edit the file of the theme.

== Installation ==

1. Upload `custom-css` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add Custom CSS in Custom CSS page in Settings menu.
